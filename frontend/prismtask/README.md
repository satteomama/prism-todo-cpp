# PrismTask — Next.js app

This is the Next.js / TypeScript application that implements PrismTask.

All user-facing documentation — install instructions, API reference,
architecture, known issues — lives in the **[root `README.md`](../../README.md)**.

## Quick commands

From this directory:

```bash
npm install
npm run dev          # http://localhost:3000
npm run build        # production build + TypeScript check
npm run lint         # ESLint
npm run typecheck    # tsc --noEmit
npm start            # serve production build
```
